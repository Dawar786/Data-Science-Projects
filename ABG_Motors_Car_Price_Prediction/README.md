# ABG Motors – Car Price Prediction

## Problem Statement
To build a predictive model to estimate car prices based on vehicle specifications to support market entry decisions.

## Dataset
Multiple CSV files containing car features such as engine size, mileage, fuel type, and horsepower.

## Approach
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building using Linear Regression and Random Forest
- Model Evaluation using R² and RMSE

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Outcome
Random Forest model achieved strong prediction accuracy and identified key pricing
